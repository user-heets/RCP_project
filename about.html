Project Description

"Rock, Paper, Scissors" Website

The website is the first stage of a larger project and represents an interactive game where the user competes against a computer opponent in the classic "Rock, Paper, Scissors" game.

The main goal of this stage is to create a fun and engaging online experience that not only allows users to play, but also demonstrates the foundation for more advanced features in the future. The site features a leaderboard, smooth animations, and a pseudo-AI opponent that analyzes your moves to provide a more challenging gameplay.

This project is open source, and you can find the full code and follow the development on my GitHub:
GitHub Repository

Project Stages

1 Website (current stage)

Development of an interactive website where users can play "Rock, Paper, Scissors" against the computer.

2 AI Model

Creation and training of an artificial intelligence model capable of recognizing hand gestures (rock, paper, scissors) in real time via camera.

3 Robotic Hand

Development of a robotic hand that will play "Rock, Paper, Scissors" with a human and always win.

Inspiration

The original idea for this project belongs to Mark Rober, who created a robotic hand that always wins at "Rock, Paper, Scissors." I was inspired by his work and decided to try to recreate this project myself.

Pseudo-AI logic for decision-making:

PHP AI Logic 
private function getAIChoice(string $game_id, string $player_name): string {
  // Get player's move history
  $stmt = $this->db->prepare(
    "SELECT user_choice
    FROM games
    WHERE game_id = ? AND player_name = ?
    ORDER BY played_at DESC
    LIMIT 10"
  );
  $stmt->execute([$game_id, $player_name]);
  $history = $stmt->fetchAll(PDO::FETCH_COLUMN);

  // If no history, pick randomly
  if (empty($history)) {
    return self::CHOICES[array_rand(self::CHOICES)];
  }

  // Analyze player patterns
  $counts = array_count_values($history);
  $most_frequent = array_search(max($counts), $counts);

  // Try to win if a pattern is detected
  $winning_moves = array_flip(self::WINNING_COMBINATIONS);
  if (max($counts) > count($history) / 3) {
    return $winning_moves[$most_frequent];
  }

  return self::CHOICES[array_rand(self::CHOICES)];
}

Functionality

  • The game consists of 10 rounds where the user plays against the computer.
  • The leaderboard displays the top 20 players by number of wins.
  • The pseudo-AI analyzes the player's move history to detect patterns.
  • If the player favors a certain move (uses it more than a third of the time), the AI chooses the move that beats it.
  • Otherwise, the AI picks randomly.

Infrastructure

  • The website is hosted on a free AWS EC2 instance running Ubuntu.
  • Apache2 web server serves both frontend and backend.
  • Deployment is done manually from a GitHub repository.
  • Frontend and backend are located in the same repository for easier deployment.

This website not only provides an entertaining experience for users but also serves as the foundation for the next stages of the project, where a real AI for gesture recognition and a robotic hand for physical interaction will be implemented.
